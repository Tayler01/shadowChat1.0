## Purpose

Track generated Shado TV assets, prompts, generated source files, optimized repo
paths, dimensions, file sizes, usage, approval state, and QA notes.

## Asset Standards

- The visual direction is Cinema Marquee: obsidian black, warm gold, black
  velvet, theater bulbs, ticket forms, poster art, and subtle analog texture.
- Assets must support mobile-first layouts and installed PWA usage.
- Only the main logo/marquee banner should require readable `SHADO TV` text.
- Most assets should be reusable and text-light or text-free.
- Avoid real brands, real platform logos, real actors, copyrighted characters,
  fake YouTube/Vimeo/TikTok marks, watermarks, or stock-looking art.
- Final static app assets belong under `public/entertainment/shado-tv/`.
- Prefer optimized WebP for app references.
- Preserve prompt detail and generated source paths so assets can be regenerated
  later.
